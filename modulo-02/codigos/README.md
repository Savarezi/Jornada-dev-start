
O código têm como objetivo demonstrar como a lógica desenvolvida em pseudocódigo e fluxogramas pode ser aplicada em uma linguagem de programação.



**Descrição:**
- Lê o valor da compra informado pelo usuário.
- Verifica se o valor é maior que **R$ 100,00**.
- Aplica **10% de desconto** quando a condição é verdadeira.
- Exibe o valor final da compra.

### Lógica utilizada

1. Ler o valor da compra.
2. Verificar se o valor é superior a R$ 100,00.
3. Aplicar desconto de 10%, quando necessário.
4. Exibir o valor final ao usuário.



> **Observação:** Este código foi desenvolvido em Harbour para fins de estudo e prática da lógica de programação.
