## O que é um ERP?

ERP significa:

**Enterprise Resource Planning**

Em português:

um sistema que centraliza toda a gestão de uma empresa.

Ele conecta áreas como:

- financeiro
- compras
- estoque
- vendas
- faturamento
- folha de pagamento
- contabilidade

Tudo conversa entre si.

Exemplo:

Quando uma nota fiscal é emitida:

✔ estoque baixa  
✔ financeiro registra  
✔ contabilidade lança  

Tudo automático.
