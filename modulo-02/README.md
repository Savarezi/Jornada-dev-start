# 📘 Módulo 02 — Lógica de Programação

Neste módulo foram desenvolvidos exercícios voltados aos fundamentos da lógica de programação, utilizando algoritmos, pseudocódigo, fluxogramas e refinamento sucessivo.

O objetivo é fortalecer o raciocínio lógico antes da implementação em linguagem Harbour/ADVPL, praticando a resolução de problemas de forma estruturada.

---

## 📚 Conteúdo

- Conceito de algoritmo
- Validação de algoritmos
- Pseudocódigo
- Fluxogramas
- Refinamento sucessivo
- Pensamento lógico
