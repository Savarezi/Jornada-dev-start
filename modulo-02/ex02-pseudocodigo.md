# Exercício 02 — Pseudocódigo

## Enunciado

Escreva em pseudocódigo um algoritmo para cada item:

**a.** Calcular a área de um retângulo (base × altura).

**b.** Verificar se um número é par ou ímpar.

**c.** Encontrar o maior entre três números.

> **Dica:** utilize as palavras **Leia**, **Escreva**, **Se...Senão** e o operador **←** para atribuição.

---

# Resolução

## a. Calcular a área de um retângulo

```text
Início

Leia Base
Leia Altura

Area ← Base * Altura

Escreva Area

Fim
```

---

## b. Verificar se um número é par ou ímpar

```text
Início

Leia Numero

Se Numero % 2 = 0
    Escreva "Par"
Senão
    Escreva "Ímpar"

Fim
```

---

## c. Encontrar o maior entre três números

```text
Início

Leia Numero1
Leia Numero2
Leia Numero3

Maior ← Numero1

Se Numero2 > Maior
    Maior ← Numero2
FimSe

Se Numero3 > Maior
    Maior ← Numero3
FimSe

Escreva Maior

Fim
```

---

## Conclusão

Os pseudocódigos apresentados seguem uma sequência lógica, possuem entrada, processamento e saída de dados, atendendo aos princípios de um algoritmo bem estruturado.
