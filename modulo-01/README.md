# Módulo 01 — Primeiros Passos com Harbour

Este módulo reúne os primeiros exercícios desenvolvidos durante a formação **Start+ TOTVS**, marcando o início da jornada na linguagem **Harbour**, base para o desenvolvimento em **ADVPL**.

## Objetivos

- Configurar o ambiente de desenvolvimento.
- Escrever os primeiros programas em Harbour.
- Compreender a estrutura básica de um programa.
- Utilizar a função `QOut()` para exibir informações na tela.
- Compilar e executar programas pelo terminal.
