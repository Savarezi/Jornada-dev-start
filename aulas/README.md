# 💻 Aulas

Esta pasta reúne todas as aulas práticas desenvolvidas durante o programa **Start+ TOTVS**.

As aulas estão organizadas por módulos e documentam minha evolução ao longo da formação, desde a configuração do ambiente até o desenvolvimento do Projeto Final utilizando **Harbour**, **ADVPL** e **TOTVS Protheus**.

Aplicar na prática os conceitos estudados durante o programa, desenvolvendo exercícios, exemplos e projetos que simulam situações reais do ambiente corporativo.
