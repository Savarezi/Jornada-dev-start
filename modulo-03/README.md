# 📘 Módulo 03 — Variáveis, Tipos e Operadores

Neste módulo foram desenvolvidos exercícios para praticar a declaração de variáveis, tipos de dados, operadores matemáticos, comparações e manipulação de dados em Harbour.

Os exercícios reforçam conceitos fundamentais para a construção de programas estruturados e servem como base para o desenvolvimento em Harbour/ADVPL.

---

## 📚 Conteúdo

- Declaração de variáveis
- Prefixos de tipo (`c`, `n`, `l`, `d`, `a`)
- Entrada de dados com `ACCEPT`
- Conversão com `Val()`
- Operadores matemáticos
- Comparação (`=` e `==`)
- Cálculos e fórmulas
- Estruturas condicionais (`IF`)
