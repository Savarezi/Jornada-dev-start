# 📝 Exercícios Extras

Esta pasta reúne os exercícios de fixação e desafios desenvolvidos durante as aulas do programa **Start+ TOTVS**.

Os arquivos desta pasta **não fazem parte das atividades obrigatórias** avaliadas automaticamente. O objetivo é praticar os conceitos apresentados em aula, reforçar o aprendizado e experimentar diferentes soluções.

---

## 🎯 Objetivos

- Fixar os conceitos aprendidos em cada aula.
- Praticar lógica de programação.
- Explorar novas soluções e abordagens.
- Desenvolver autonomia na resolução de problemas.


---

> 🚀 A prática constante é o caminho para evoluir como desenvolvedor. Cada exercício resolvido representa mais experiência e confiança na programação.
