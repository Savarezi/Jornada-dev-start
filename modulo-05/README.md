Este diretório reúne os exercícios desenvolvidos durante o **Módulo 05** da **Jornada DEV START (TOTVS Paulista)**.

## 🎯 Objetivos de aprendizagem

- Utilizar estruturas **FOR/NEXT**.
- Trabalhar com **WHILE** em repetições indefinidas.
- Criar contadores e acumuladores.
- Validar entradas utilizando loops.
- Desenvolver programas interativos com repetição.
- Aplicar lógica de programação em desafios.
