# 📘 Módulo 04 — Decisões (Condicionais)

Neste módulo foram desenvolvidos exercícios para praticar estruturas de decisão em Harbour, utilizando `IF`, `ELSEIF`, `ELSE`, `DO CASE` e `OTHERWISE`.

Os exercícios abordam situações do dia a dia, como comparações, cálculos, reajustes salariais, calculadora, identificação de meses e plano de saúde.

---

## 📚 Conteúdo

- Estruturas condicionais (`IF`, `ELSEIF`, `ELSE`)
- Estrutura `DO CASE`
- Operadores relacionais
- Operadores lógicos
- Comparação de valores
- Validação de dados
- Cálculos condicionais
