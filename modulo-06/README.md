## 🎯 Objetivos de aprendizagem

- Criar e utilizar funções.
- Diferenciar funções de procedimentos.
- Dividir um programa em módulos menores.
- Criar funções reutilizáveis.
- Utilizar arrays unidimensionais.
- Trabalhar com arrays multidimensionais.
- Utilizar `AADD()` para adicionar elementos.
- Percorrer arrays utilizando `FOR`.
- Ordenar arrays utilizando `ASORT()`.
- Implementar o algoritmo Bubble Sort manualmente.
- Integrar funções, arrays e estruturas de repetição.
- Organizar bibliotecas externas utilizando `SET PROCEDURE TO`.
